# BandBook

> A REST API to serve the band-management app BandBook

## Introduction

BandBook lets bandmembers easily schedule band related events and get in contact in message-threads.

## Features

- Login with Google
- Forum feature
- Calendar feature

## Technologies Used

- Jetpack Compose
- Kotlin
- Gradle
