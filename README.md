# BandBook

> A band-management app for all musicians

## Introduction

BandBook lets bandmembers easily schedule band related events and get in contact in message-threads.

## Features

- Login with Google
- Forum feature
- Calendar feature

## Technologies Used

- Jetpack Compose
- Kotlin
- Gradle

## Getting Started

Clone this repository and run main() from Application.kt

### Prerequisites

Android Studio
